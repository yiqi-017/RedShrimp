# RedShrimp 小红虾

> 服务于 [GenericAgent](../GenericAgent/) 的小红书流量技能模块。

## 这是什么

小红书 MCP 能发帖、能看帖，但不知道 **发什么、怎么发、什么时候发**。

RedShrimp 要解决的就是这一层 —— 让 GenericAgent 拥有「流量直觉」：

- **爆款分析** — 什么样的帖子容易火？标题、封面、正文、标签的共性规律
- **热点追踪** — 当前平台在推什么话题、什么赛道有流量红利
- **养号策略** — 新号冷启动路径、互动节奏、权重提升手段
- **平台机制** — 推荐算法逻辑、流量池层级、限流/违规的边界

## 与 GenericAgent 的关系

```
GenericAgent（决策 + 执行）
  ├── 小红书 MCP（发帖/看帖的手）
  └── RedShrimp（流量策略的脑）
```

GenericAgent 负责 agent loop 和工具调用，RedShrimp 提供流量侧的知识与策略，最终目标是 **自动发帖起号**。
