# 核心指标权重参考 (CES 模型)

| 互动类型 | 算法权重系数 | 运营建议 |
| :--- | :---: | :--- |
| 点赞 (Like) | 1 | 基础指标，封面和标题决定 |
| 收藏 (Save) | 1 ~ 2 | 核心指标，内容必须具有干货、清单或“日后必用”属性 |
| 评论 (Comment) | 4 | 流量放大器，文案末尾抛出问题或故意留白引战（温和） |
| 转发 (Share) | 4 | 社交货币，能够帮用户表达态度的内容 |
| 关注 (Follow) | 8 | 最高权重，人设吸引力与主页整体调性的综合体现 |

# 标题爆款公式库
1. **人群定向 + 痛点 + 解决方案:** “租房党必看！300元搞定极简原木风卧室”
2. **反常识/避坑:** “不要瞎买！这5个网红收纳神器实际上是智商税”
3. **数字清单:** “吐血整理！去XX旅游必去的10个小众打卡地”
4. **情感共鸣:** “25岁裸辞，我后悔了吗？”
