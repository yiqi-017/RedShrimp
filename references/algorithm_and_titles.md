# 核心指标权重参考 (CES 模型)

| 互动类型 | 算法权重系数 | 运营建议 |
| :--- | :---: | :--- |
| 点赞 (Like) | 1 | 基础指标，封面够美、标题够吸睛就能拿到啦。 |
| 收藏 (Save) | 1 ~ 2 | 核心指标，内容必须全是干货、清单，或者能让人觉得“以后肯定用得上”。 |
| 评论 (Comment) | 4 | **绝对的流量放大器！** 文案结尾一定要抛问题或者故意留破绽，让大家来吵（温和地吵）。 |
| 转发 (Share) | 4 | 社交货币，内容得能帮姐妹们表达态度，或者适合转给男朋友看/转到闺蜜群。 |
| 关注 (Follow) | 8 | 最高权重！这靠的是你的人设魅力和主页整体调性，要让人觉得“哇这博主我好爱”。 |

# 标题爆款公式库
1. **人群定向 + 痛点 + 解决方案:** “租房党/早八人/干皮姐妹进！300元搞定极简原木风卧室，绝绝子”
2. **反常识/避坑:** “听句劝千万别瞎买！这5个风很大的网红神器其实全是智商税❌”
3. **数字清单:** “压箱底吐血整理！去XX旅游必去的10个宝藏小众打卡地”
4. **情感共鸣/焦虑贩卖:** “25岁裸辞在家躺平第30天，我到底在焦虑什么？”
