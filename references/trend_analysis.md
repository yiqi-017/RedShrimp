# 小红书趋势分析与爆款预测 (Trend Analysis)

## 概述
当用户“不知道发什么”或没有明确的赛道时，我们需要利用小红书的实时推荐流和搜索热词，来反向推导当下最有“爆火潜质”的赛道和话题。

我们通过调用本地部署的 **小红书 MCP (xiaohongshu-mcp)** 工具，可以实时获取平台的推荐列表 (Feed) 和搜索结果 (Search)，从而进行趋势分析。

## 趋势侦测步骤 (利用 MCP)

如果用户让你“推荐最近火的赛道”，请按以下步骤调用小红书 MCP 工具（确保 MCP 服务已在本地运行）：

### 1. 获取首页推荐流 (Feed Analysis)
首页推荐流代表了平台当前正在强推的泛流量池内容。
- **操作:** 调用 MCP 工具获取首页推荐列表。
- **分析维度:** 
  - 观察前 20 篇推荐笔记，统计出现频率最高的赛道（如：最近是不是满屏的“多巴胺穿搭”或者“极简护肤”？）。
  - 提取高频出现的**情绪词**和**痛点词**。

### 2. 搜索框热词侦测 (Search Trends)
搜索框的下拉词和默认推荐词，代表了用户当前最强烈的刚需。
- **操作:** 调用 MCP 的搜索功能，输入一些泛生活关键词（如“打工人”、“周末”、“搞钱”），观察返回的高赞笔记。
- **分析维度:** 
  - 寻找那些发布时间极短（如 24 小时内）但点赞量异常高的“黑马笔记”。这种笔记代表了一个正在爆发的新趋势。

### 3. 数据提炼与赛道推荐
综合 Feed 流和搜索结果，向用户推荐 2-3 个最具爆发力的具体切入点。推荐必须具体到“话题”而不是泛泛的“赛道”。

**错误示范:** “建议你做美食赛道。”
**正确示范:** “宝子，我刚看了一下现在的流量趋势，最近【减脂宿舍党免烤箱甜品】这个话题简直杀疯了，好几篇新笔记点赞都破万了！咱们可以马上切入这个细分赛道！”

## 常见常青树与易爆小趋势参考 (Fallback)
如果 MCP 暂时无法调用或获取数据失败，可以向用户推荐以下几个永远自带流量的“黄金切口”：

1. **“低门槛搞钱”:** 例如“下班后的 2 小时副业实操”、“零基础怎么做自媒体”。
2. **“打工人发疯日常”:** 利用夸张的情绪宣泄职场焦虑。
3. **“极端反差逆袭”:** 比如“三个月从土妞到财阀千金感”。
4. **“无脑听劝系列”:** “听劝！别买XX，买这个” —— 这种永远能骗到点击率。
